# Apple Financil Analysis

# Task Overview

This task is a Power BI dashboard created to analyze and visualize key business metrics. The dataset includes financial, operational, and sales data. The goal of this project is to derive actionable insights and present them in an interactive, easy-to-understand format.

# Key Objectives:

Analyze financial performance (Revenue, Profit, EBITDA, Net Income).

Track operational metrics (Employees, Cash on Hand, Assets).

Visualize trends, comparisons, and top-performing categories.

Provide actionable insights for decision-making.

# Dataset

The dataset used in this project contains the following columns:

Cash on Hand

EBITDA (Million)

Employees

EPS

Gross Margin

Gross Profit

Long Term Debt

Net Income

Operating Income

PE Ratio

Revenue

Shares Outstanding

Total Assets

# Analysis & Visualizations

The dashboard includes:

Financial Overview: Total Revenue, Net Income, EBITDA trends.

Profitability Analysis: Profit Margin calculations and comparisons.

Top 5 Analysis: Top 5 products, departments, or regions based on revenue and profit.

Employee & Operational Insights: Analysis of employees vs performance metrics.

Interactive Charts: Bar charts, line charts, pie charts, and slicers for dynamic filtering.


# Insights

Some key insights derived from the analysis:

Revenue and net income trends over time.

Most profitable categories and underperforming areas.

Correlation between operational metrics (employees, assets) and financial performance.

# Tools & Technologies

Power BI Desktop – for building dashboards and reports

DAX (Data Analysis Expressions) – for calculated metrics

Excel – for data preprocessing

# Learning Outcomes

Through this project, I learned:

How to clean and prepare data for visualization in Power BI.

Creating interactive dashboards with meaningful visualizations.

Calculating business metrics like profit margin and EBITDA using DAX.

Drawing insights and presenting them effectively for decision-making.
